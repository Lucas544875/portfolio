# VAPORWAVE — レイマーチ・グリッドの夕暮れ

ヴェイパーウェイブの定番ビジュアル(ネオングリッド・ストライプの太陽・ワイヤーフレームの山岳)を
レイマーチングで再現し、ノイズとインタラクションを加えたエモーショナルな画面のプロトタイプ。
他の `prototype-*` と同じく、フレームワーク無し・ビルド不要の単体 WebGL2 アプリ。

## 実行方法

```sh
cd prototype-vaporwave
python3 -m http.server 8080
# → http://localhost:8080 をブラウザで開く
```

## できていること

- **レイマーチング**: fbm ノイズのハイトフィールド(中央は平原、左右に山岳)を
  レイマーチ + 2分法リファインで描画。ライティングは行わず、
  グリッドのエミッシブ + 距離フォグ + 高さティントだけのネオン構成
- **空**: グラデーション + ストライプ入りの太陽(下端ほど欠けが強い)+ 瞬く星を
  レイ方向からプロシージャル生成。太陽はゆっくり明滅する
- **VHS ポストパス**: シーンを低解像度 FBO(0.66x)に描き、フルスクリーンパスで
  バレル歪み・色収差・走査線・行ジッタ(たまにバースト)・トラッキングバー・
  グレイン・ビネットを乗せる
- **インタラクション**:
  - 画面をなぞる / クリック → スクリーン座標をグリッド平面へアンプロジェクトし、
    その場所から波紋(高さの変位 + 波面の発光リング)が伝わる
  - ポインタ移動 → 視点のパララックス
  - ホイール → 前進速度の加減速
  - ボタン → VHS ノイズの ON/OFF、パレット切替(DUSK / MIDNIGHT / DAWN、色は補間で遷移)
- ページを開くと自動で波紋が数回落ちて、触り方を提示する
- イージング(視点・速度・ノイズ量・パレット)はすべて `1 - exp(-rate * dt)` の
  時間ベースで、フレームレートに依存しない

## 構成

- `index.html` — キャンバスとトグルボタン、ヒント
- `main.js` — シェーダ2本(シーン / ポスト)と入力処理をすべて含む
- `style.css` — UI のスタイル

## 既知の割り切り

- カメラの前進距離が非常に大きくなると(数十分オーダー)ノイズのハッシュ精度が
  落ちて地形が粗くなる。プロトタイプとして許容
- 波紋のワールド座標計算(JS 側アンプロジェクト)はグリッド平面 y=0 との交点で
  近似しており、山肌をなぞった場合は少し手前にずれる
