# ポートフォリオサイト コンセプトメモ

作成日: 2026-07-11(Claude Codeとのブレストを整理)

## コンセプト:「結露したガラスの向こう側」

雨の夜、曇ったガラス越しに世界を見ている——というサイト全体の設定。
メモしていたキーワード(流体シミュレーション / 雨で曇ったガラス / 指でなぞった文字 / KanjiVG / 階層を下るスライド / ホラー表現)を「ガラス・水・文字」という一本の筋で統合する。

---

## 1. ヒーロー(トップ画面)

- 画面全体が結露で曇ったガラス。奥に自作作品のシーンがぼんやり透けて見える
- 水滴シェーダーが上から流れ落ちる(参考: WebGL総本山の記事のデモ)
- **KanjiVGの筆順データを使い、名前の漢字が「指でなぞられるように」一画ずつ書かれていく**
  - なぞられた軌跡だけ結露が拭われ、奥の景色が鮮明に見える
- マウス / タッチでもガラスを自由に拭える(第一印象のインタラクション)

> KanjiVGは筆順つきの漢字SVGパスデータ。「正しい筆順で指がなぞる」演出がデータ駆動で作れる。日本人ならではの武器になる。

## 2. Stable Fluids の使いどころ

- 拭った跡・カーソルの動きを速度場として流体シミュレーションに注入
- 結露の曇り(dye)が流体的に揺らぎながら戻ってくる
- → 「水滴」「拭き文字」「流体」を1つのシミュレーションに統合できる
- ページ遷移時に、画面全体へインクのように色が流れ込むトランジションにも流用可能

## 3. ナビゲーション:「沈んでいく」階層構造

- 「階層を下ると下にスライド」を、水中 / ガラスの層を降りていく演出として解釈する
- 深くなるほど暗く、曇りが濃く、水圧感が増す
- About → Works → 各作品詳細、と降りるたびに被写界深度と色温度が変化し「深度」を体感させる

## 4. ホラー表現:「静かな不穏さ」に留める

本格ホラーは採用担当者を選ぶため、イースターエッグ的な仕込みにする。

- 放置していると、拭いていないガラスの端に誰かが文字を書いた跡がうっすら浮かぶ
- 最下層(Contactなど)だけ、ガラスの向こうを一瞬何かの影がよぎる
- 「気づいた人だけゾッとする」設計で、技術力と遊び心の両方をアピール
- 余裕があったら実装することとし、一旦保留

## 技術スタック(目安)

| 要素 | 手段 |
|---|---|
| 描画 | Three.js(または軽量なOGL)+ ping-pong FBO で Stable Fluids |
| 流体の参考実装 | Pavel Dobryakov「WebGL-Fluid-Simulation」(GPU実装の定番) |
| 結露・水滴 | 法線マップで背景を屈折 + ブラーで曇り表現。曇り度は流体simのdyeテクスチャで制御 |
| 文字 | KanjiVGのパスを筆順に沿って読み、ブラシテクスチャをFBOに描き込む |
| スクロール | Lenis等の慣性スクロール + セクションスナップで「層を降りる」感 |

### 注意点

- モバイルでは流体simの解像度を1/4程度に落とすフォールバックが必須
- **「エフェクトが主役になりすぎて作品が見えない」がこの手のサイトの典型的な失敗**。Works一覧以降はエフェクトを引き算する設計にする
